# todo_list
STORESのVue勉強会のためのリポジトリ

セットアップ
```ruby
npm install -g vue-cli
npm install bulma
```

サーバー起動コマンド
```ruby
npm run server
```

ローカルでの動作確認
```ruby
http://127.0.0.1:8080/
```
